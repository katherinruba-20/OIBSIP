#OIBSIP

Task 1: ATM Interface 💳

DESCRIPTION:
This project simulates an ATM Interface system using Java. 
It allows users to perform basic banking operations like deposit, withdraw, transfer, and view transaction history after login.

FEATURES:
- User authentication (User ID & PIN)
- Deposit money
- Withdraw money
- Transfer money
- Transaction history
- Balance display

TECHNOLOGIES USED:
- Java
- OOP Concepts (Basic)

 TASK 2: Number Guessing Game 🎯

DESCRIPTION:
This project is a simple Number Guessing Game developed using Java. 
The program generates a random number between 1 and 100, and the user has to guess the correct number within a limited number of attempts.

FEATURES:
- Random number generation
- User input using Scanner
- Hint system (Too High / Too Low)
- Limited attempts (5 chances)
- Displays result (Win / Game Over)

TECHNOLOGIES USED:
- Java
- OOP Concepts (Basic)
